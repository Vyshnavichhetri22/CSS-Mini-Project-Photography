# CSS Mini Project - Photography

This is a mini front-end project built using HTML and CSS.  
It includes a responsive sidebar menu, toggle buttons, and social media links.

## Features
- Responsive sidebar menu
- Toggle button to open/close sidebar
- Menu links with Font Awesome icons
- Social media section
- Google Fonts (Poppins)
- HTML & CSS only (no JS)

## Technologies
- HTML5
- CSS3
- Google Fonts
- Font Awesome
